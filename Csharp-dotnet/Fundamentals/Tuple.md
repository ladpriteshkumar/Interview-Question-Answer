## What is Tuple ?   
A tuple in C# is a lightweight data structure that lets you group multiple values into a single object without creating a custom class or struct. Each value in the tuple can have a different type, and the tuple preserves the order of those values.

## Why tuples are useful   
- Returning multiple values from a method
- Grouping related values without defining a class
- Improving readability with named elements
- Quick, temporary data structures
